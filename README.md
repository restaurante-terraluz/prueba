# Terraluz

Sitio web de presentación del proyecto de interiorismo **Terraluz**, un restaurante cafetería gourmet con estética mediterránea contemporánea, basada en la combinación de materia y luz.

Es un sitio estático multipágina que documenta el desarrollo del proyecto: concepto, paleta cromática, materiales, iluminación, espacio y programa, instalaciones, render y referencias de inspiración.

## Estructura

```
index.html            Página principal (concepto y presentación)
inspiracion/          Referencias e inspiración
materiales/           Materiales y acabados
renders/              Renders del proyecto
espacio/              Espacio y programa funcional
iluminacion/          Diseño de iluminación
instalaciones/        Instalaciones técnicas
documentacion/        Documentación del proyecto
css/, js/, img/       Estilos, scripts y recursos gráficos
```

## Uso

Sitio estático, sin build ni dependencias. Para verlo en local, abrir `index.html` en el navegador o servirlo con cualquier servidor estático:

```bash
npx serve .
```

## Despliegue

Pensado para publicarse como GitHub Pages u otro hosting estático equivalente.
